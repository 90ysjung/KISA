
Segugio - Efficient Behavior-Based Tracking of Malware-Control Domains in Large ISP Networks

1. 요약 

대규모 ISP 네트워크에서 새로운 악성 프로그램 제어 도메인 이름발생을 
효율적으로 추적 할 수있는 시스템 제안한다
DNS 트래픽을 수동적으로 모니터하여 기계 도메인을 구축후
누가 무엇을 쿼리하는지 정보를 수집 후 알려진 노드를 레이블링합니다.
양성 또는 맬웨어 관련 문제를 해결하기 위해
이전에 알려지지 않은 맬웨어 제어 도메인을 정확하게 탐지합니다.


2. Intro
광범위한 연구 노력에도 불구하고 악성 코드가 여전히 광범위하며
멀웨어 감염은 계속 증가하고 있습니다. 
때문에 악성 코드는 오늘날 사이버 범죄의 근원입니다.
그것을 극복하기 위한 대안도  지속하는 발전하는 것이 매우 중요합니다.
멀웨어 제어(C & C (Command-and-Control)) 추적 하여
악성 프로그램과 관련된 도메인 일 가능성이 높다는 결론을 내릴 수 있으며
이를 기계학습으로 자동 예측하는 실험을 한다

향후 악성 프로그램 제어 도메인(악성도메인) 예측기를 구현한다


4. 연구
알려진 멀웨어에 우선 레이블을 지정할 수 있습니다.
C & C 도메인에 영향을 미치고 결과적으로 해당 레이블을 전파합니다.
C & C를 조회하는 기계를 표시하여
도메인을 맬웨어에 감염된 것으로 간주합니다. 유사하게, 우리는 양성으로 분류 할 수있다.
효율적인 분류를 목표로합니다.

주요 목표는 DNS 쿼리 동작을 추적하는 것입니다.
현재의 맬웨어에 감염된 시스템의 새로운 발견
악성 프로그램 제어 도메인

밀웨어에 감염된 컴퓨터와 DNS query 정보 관계를 분석
악성 코드 제어를 쿼리하는 경우 시스템을 맬웨어로 분류합니다.


5. Data(Training set)
- IP addresses
- Domain activity(query)
- DNS 정보(trafic)
